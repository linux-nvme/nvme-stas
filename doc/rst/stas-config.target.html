Name

stas-config.target

Synopsis

/usr/lib/systemd/system/stas-config.target

Description

This target is used as a synchronization point before starting stacd.service(8) and stafd.service(8).

It ensures that /etc/nvme/hostnqn and /etc/nvme/hostid are present before starting stacd.service(8) and stafd.service(8).
