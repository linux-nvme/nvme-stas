Name

stas-config@.service

Synopsis

/usr/lib/systemd/system/stas-config@.service

Description

This service is used for the automatic run-time generation of NVMe configuration located in /etc/nvme (e.g. /etc/nvme/hostnqn). This is needed by stacd.service(8) and stafd.service(8).
